# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [4.2.0] - 2025-12-27

### Added
- Complete Svelte + Vite portfolio implementation
- OKLCH color system with light/dark theme support
- DTCG v2025.10 compliant design tokens
- Theme toggle with system preference sync
- Skip-to-content link for accessibility
- Focus-visible states for keyboard navigation
- prefers-reduced-motion support
- AI agent detection with enhanced JSON-LD
- PWA manifest for installability
- SEO with Open Graph and Twitter Card meta tags

### Changed
- Accent colors adjusted for WCAG AA 4.5:1 contrast compliance
- Content max-width optimized for 50-75 character line length
- Caption colors use solid OKLCH values instead of opacity

### Fixed
- Connect section links maintain underline on hover
- Design tokens synced with CSS custom properties
