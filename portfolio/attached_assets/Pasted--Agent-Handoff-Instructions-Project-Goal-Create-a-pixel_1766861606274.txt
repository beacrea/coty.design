# Agent Handoff Instructions

## Project Goal

Create a pixel-perfect 1:1 clone of [coty.design](https://coty.design) portfolio using:
- **Svelte** with Vite (NOT React)
- **DTCG v2025.10** compliant design tokens
- **OKLCH color space** for perceptual uniformity
- Light/dark theme support
- Mobile-optimized responsive design (sm/md/lg breakpoints)
- PWA capabilities
- SEO with Open Graph and JSON-LD structured data
- AI agent detection for enhanced content delivery

---

## Critical Requirements

1. **Use Svelte** - This is non-negotiable. Create project from Svelte template.
2. **OKLCH Colors** - All colors must use oklch() for perceptual uniformity
3. **DTCG Tokens** - Follow v2025.10 specification for all design tokens
4. **Red Hat Mono** - Typography uses this font with weights 400/460/680

## Theme Toggle Design

**Dark Mode (Moon Icon):**
- Border: `oklch(1 0 0 / 0.3)`
- Fill: `oklch(0.521 0.215 277)` (violet)
- Stroke: `oklch(0.576 0.213 277)`
- Rays: `oklch(0.843 0.098 277)`

**Light Mode (Sun Icon):**
- Border: `oklch(0 0 0 / 0.15)`
- Fill: `oklch(0.495 0.160 40)` (orange)
- Stroke: `oklch(0.576 0.180 45)`
- Rays: `oklch(0.816 0.130 55)`

## Key Implementation Notes

1. **Theme Toggle Position**: Fixed, top-right, offset by page padding
2. **Bullet Lists**: Use circular bullets (8px) with vertical offset to align with text
3. **Links**: Underlined with 3px offset, remove underline on hover
4. **Sections**: "Recent Focus" and "Domain Expertise" use bullet lists; "Current Status" and "Foundations" use paragraph style with inline bullets
5. **Transitions**: 300ms for theme colors, 200ms for toggle animation
6. **Mobile-first**: Start with sm values, use media queries for md/lg

## Quick Start for New Agent

1. Create new Svelte project with Vite
2. Install: `tailwindcss`, `isbot`, font link for Red Hat Mono
3. Copy design tokens to CSS (see OKLCH colors above)
4. Create components following the structure above
5. Implement theme toggle with localStorage persistence
6. Add SEO meta tags and JSON-LD
7. Add PWA manifest
8. Test responsive breakpoints and theme switching
